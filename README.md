# Flower Classification ML Project

This is a beginner Machine Learning classification project built using Python and Scikit-learn.

## Project Overview

The goal of this project is to classify iris flowers into different species using machine learning.

The model predicts the flower species based on:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

## Technologies and Libraries Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

## Machine Learning Algorithm Used

* K-Nearest Neighbors (KNN)

## Concepts Learned

* Data Loading and Preprocessing
* Train-Test Split
* Model Training
* Prediction
* Accuracy Evaluation
* Data Visualization

## Files Included

* `flower_classification.ipynb` → Jupyter Notebook
* `flower_classification.py` → Python Script
* `README.md` → Project Documentation

## Outcome

The model was successfully trained and tested on the Iris dataset and achieved good classification accuracy.
